Out Of Order

A "so bad its good" parody retro FPS throwback game I made for the "So Bad It's Good" game jam on itch.io back in 2019. In fact, my submission came in first place.

Out Of Order was my first ever game jam submission, and the first game I've actually put online. Learned a lot from making it, it was the first game I completed with Unity 3D, and my first game with actual AI. However poor it may be...

Features:

  • Pre-rendered 3D models turned into sprites
  • A partially functioning stealth system
  • Boss fight
  • Multiple weapon types
  • Multiple enemy types
  • 3D rendered cutscenes (this seemed to be the most popular feature)

Ja Wizardman

A terrible first person action game based off of someone else's terrible parody of the Witcher franchise.

The most mechanically complicated project I've worked on so far. Features include:

  • Branching dialogue trees
  • Multi-directional melee combat (worked better than expected)
  • Faction and opinions tracked between the player and between NPC's
  • Multiple endings
  • Poorly detailed small open world
